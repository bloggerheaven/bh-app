# Development

## Requirements

* [Node](https://nodejs.org) (install it with [nvm](https://github.com/creationix/nvm))

## Installation

```
npm i
```

## Run

### Browser

```
ionic serve -l
```

### Emulate

```
ionic emulate ios
```

## Build

```
ionic build ios
ionic build android
```

## Test

```
npm test
```
